# jetbot_ros
ROS nodes and Gazebo model for NVIDIA JetBot with Jetson Nano
origin:https://github.com/dusty-nv/jetbot_ros

# 添加功能
运行：
rosrun jetbot_ros jetbot_motors.py

再开启另一个终端运行：
rosrun jetbot_ros jetbot_teleop.py

w:forward
d:backward
a:left
d:right
e:stop
